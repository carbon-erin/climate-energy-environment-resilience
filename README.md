# Formation Concept

One-page website for an early-stage nonprofit formation concept — a public-interest environmental and energy information institute.

## Local preview

Open `index.html` in a browser, or serve the folder locally:

```bash
# Python
python -m http.server 8080

# Node
npx serve .
```

Then visit `http://localhost:8080`.

## GitHub Pages

1. In the repository **Settings → Pages**, set **Source** to **Deploy from a branch**.
2. Choose branch `main` and folder `/ (root)`.
3. Save. The site will be available at the Pages URL after a short deploy.

Files: `index.html` (page), `styles.css` (styles). No build step required.
